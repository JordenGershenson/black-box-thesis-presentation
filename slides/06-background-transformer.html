Background: The Transformer Architecture

The Core Idea: Self-Attention

To predict the weather at location X, which other locations matter?

Self-attention learns which locations are relevant — assigning weights to each connection

Key Properties

✓ Global Context

Every position can attend to every other position — captures long-range dependencies

✓ Learned Relevance

Attention weights are learned from data — model discovers what matters

✗ Quadratic Complexity

O(N²) — comparing every pair of positions becomes intractable for large inputs

Revolution: Transformers power GPT, BERT, Vision Transformers, and now weather models

THE SCALE PROBLEM FOR WEATHER

A global weather grid has 721 × 1440 = 1 million+ locations. Standard attention would require 1012 computations per layer — impossible.
